IaSdkConfigServerEnvironment

IaSdkConfigServerEnvironment enum

Identifier for the ia.de server environments.

3 different server environments are set up: DEV, QA, and PROD.

The development and staging environments are set up for internal testing and are accessible with VPN only, while the production environment is the production-ready environment to which the end-users have access.

VPN access can be provided by the ia.de team on inquiry.
